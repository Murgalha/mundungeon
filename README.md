# mundungeon
Rogue-like dungeon game

## Dependencies

### Zig
This project requires `zmath` to run, just place it in the `deps` folder
and you're good to go.

### C
`freetype2` and `SDL2`, alongside with OpenGL are required to build and run it, although no additional effort is required
after they are installed, since `build.zig` already does all the work.

## Run
Run `zig build run` to compile and run the game.
